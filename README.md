#### Clone a folder
The "correct order" to push the code is:
```
git commit
git pull
git push
```

#### Update a repo
It is recommend to update  
Whenever a new file is created in the repo. We have to add the file into the tracking system  
```
git add <filenames>
```
or add all files by
```
git add .
```
